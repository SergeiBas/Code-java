# Біатлон — облік результатів змагань

Курсова робота: веб-застосунок для ведення бази даних з результатами змагань з біатлону (країни, спортсмени, дисципліни, змагання, протоколи).

## Технології

- **Java 17**, **Spring Boot 3.3**
- **Thymeleaf** — серверний HTML-інтерфейс
- **SQLite** — локальна БД (`data/biathlon_java.db`)
- **JdbcTemplate** — робота з SQL без JPA

## Можливості

- CRUD для країн, спортсменів (Ч/Ж), змагань і дисциплін
- Внесення результатів з урахуванням правил штрафів IBU:
  - індивідуальна гонка (+1 хв / +45 с за промах);
  - спринт, переслідування, мас-старт (штрафне коло 25 с);
  - естафети (командний час + етапи 1–4)
- Автоматичний розрахунок підсумкового часу та місць
- Естафети: командний залік по країні, під етапами — кожен спортсмен
- Залік балів Кубка світу (90…1 за місцем), фільтри та сортування списків
- Звіти за змаганням і дисципліною

## Запуск

**Потрібно:** JDK 17+, Maven 3.9+

```bash
git clone <url-репозиторію>
cd Code-java

mvn spring-boot:run
```

Відкрити в браузері: [http://localhost:8080](http://localhost:8080)

При першому старті (порожня БД) виконуються `src/main/resources/db/schema.sql` і `seed.sql`.

Щоб скинути демо-дані:

```bash
rm -f data/biathlon_java.db
mvn spring-boot:run
```

Альтернатива — збірка JAR:

```bash
mvn -DskipTests package
java -jar target/biathlon-app-1.0.0.jar
```

## Структура проєкту

```
src/main/java/com/biathlon/app/
  BiathlonApplication.java   — точка входу
  WebController.java         — HTTP та сторінки
  DbInitializer.java         — ініціалізація БД
  ResultLogic.java           — розрахунок часу/штрафів
  PlaceLogic.java            — місця (індивідуально / естафета)
  PointsLogic.java           — бали Кубка світу
  ResultsGrouping.java       — групування результатів у UI
  AthleteListHelper.java     — списки спортсменів і фільтри

src/main/resources/
  db/schema.sql, db/seed.sql
  templates/                 — HTML (Thymeleaf)
  static/style.css
```

## База даних

| Таблиця       | Призначення                          |
|---------------|--------------------------------------|
| `country`     | країни (ISO, назва)                  |
| `athlete`     | спортсмени, стать M/W                |
| `competition` | змагання, місце, дати, сезон         |
| `discipline`  | формат гонки, штрафи, стать          |
| `result`      | результат старту (час, промахи, місце) |

## Сторінки

| URL            | Опис                    |
|----------------|-------------------------|
| `/`            | головна                 |
| `/countries`   | країни                  |
| `/athletes`    | спортсмени              |
| `/competitions`| змагання                |
| `/disciplines` | дисципліни              |
| `/results`     | результати та залік     |
| `/reports`     | звіт по змаганню        |
